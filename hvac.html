Use when: System uses fixed metering device (not TXV at evaporator).

  • Measure suction pressure + suction line temp at outdoor unit.
  • Convert suction pressure to saturation temp (PT chart for refrigerant).
  • Superheat = Suction Line Temp - Saturation Temp.
  • Compare measured superheat to target superheat chart (indoor WB + outdoor DB).

Interpretation:

  • Measured SH high vs target: likely undercharge or starved evaporator.
  • Measured SH low vs target: likely overfeed/overcharge/flooding risk.

Adjustment rule: Add/remove refrigerant in small increments, let system stabilize, recheck.

Always verify airflow before adjusting charge.

Use when: Evaporator has TXV/EEV. Charge by subcooling target on data plate/manual.

  • Measure liquid pressure + liquid line temp at outdoor unit.
  • Convert liquid pressure to saturation temp.
  • Subcooling = Saturation Temp - Liquid Line Temp.

Interpretation:

  • Low SC vs target: undercharge likely.
  • High SC vs target: overcharge or liquid backed up in condenser.

Field baseline: Many systems land near 8–12°F, but manufacturer target wins.

Do not charge to “rule of thumb” if OEM gives explicit target.

TESP = Return Static + Supply Static (magnitude values, in. w.c.).

  • Take readings in proper test ports near air handler/furnace cabinet.
  • Compare to blower’s rated max external static (nameplate/install manual).

Quick fault cues:

  • High return static: dirty filter, undersized return, blocked return path.
  • High supply static: dirty coil, closed dampers, undersized supply ducts.
  • Total high static: airflow low, comfort and coil performance degraded.

Airflow/charge/Delta-T diagnostics are unreliable until static issues are corrected.

  • Measure return and supply dry bulb close to equipment (same side of coil/furnace boundaries each time).
  • Delta-T = Return Air Temp - Supply Air Temp (cooling mode).

Cooling quick reference:

  • ~16–22°F often seen in many properly operating comfort systems.
  • Low split: possible high airflow, low charge, high latent load, duct leakage, or mixed air issues.
  • High split: possible low airflow (dirty filter/coil), restriction, or low indoor load conditions.

Delta-T is a screening metric, not a standalone diagnosis.

Capacitor:

  • Test µF with power off + safely discharged capacitor.
  • Replace if outside tolerance (commonly ±6% unless marked otherwise).

Contactor:

  • Verify coil control voltage on call.
  • Check line/load voltage drop across contacts when closed.
  • Inspect pitted/burned contacts.

Amp draw:

  • Compare compressor/fan amps to nameplate ratings (RLA/FLA context).
  • High amps + high head often indicates airflow/charge/condenser issues.

Lockout-tagout discipline and meter safety are non-negotiable.

Observed Pattern Likely Causes First Checks
Low suction + low head Undercharge / evaporator starvation Leak search, SH/SC validation, airflow baseline
Low suction + high head Metering restriction / airflow problem Filter/coil/static, line temp drop across restriction points
High suction + low head Compressor inefficiency / valve issue Compression ratio, amp draw, discharge temp trend
High suction + high head Overcharge / non-condensables / heavy load Condenser airflow, SC trend, ambient/load context

Never diagnose by pressures alone—pair with SH, SC, airflow, and electrical data.

  1. Call for heat from thermostat.
  2. Inducer motor starts.
  3. Pressure switch proves draft.
  4. Igniter energizes (HSI/spark).
  5. Gas valve opens.
  6. Flame sensor proves flame.
  7. Blower starts after delay.
  8. Unit runs until setpoint; post-purge cycle.

Common fail points:

  • Pressure switch not closing (venting, tubing, inducer performance).
  • Igniter failure (open element / no voltage).
  • Flame sensor contamination (microamp signal weak).
  • Limit trips from low airflow/high temp rise.

Use fault code + meter readings together before replacing parts.

Pre-start:

  • Model/serial documented, filter clean, coil clear, drains confirmed.
  • Line set size/length validated, evacuation and decay test completed.
  • Electrical tightness/voltage verified.

Startup readings to capture:

  • Outdoor ambient, indoor return/supply dry bulb (and WB if available).
  • Suction/liquid pressures + line temps.
  • Superheat + subcooling.
  • Voltage/amperage (compressor, outdoor fan, indoor blower).
  • Static pressure and airflow setting.
  • Delta-T and condensate performance.

Closeout:

  • Set airflow speed tap/profile correctly.
  • Panel/door secure, insulation/seals restored.
  • Thermostat setup verified, customer orientation done, photos uploaded.

No Cool

  1. Confirm thermostat call + settings.
  2. Verify indoor/outdoor power and contactor pull-in.
  3. Check airflow baseline (filter, blower, coil, static).
  4. Capture pressures, SH/SC, delta-T, amps.
  5. Classify root cause: airflow / charge / electrical / control.

Freeze-Up

  1. Shut cooling, run fan to thaw.
  2. Check airflow restriction first.
  3. Then evaluate charge with SH/SC.
  4. Rule out blower speed/control faults.

No Heat (Gas Furnace)

  1. Read fault code.
  2. Follow sequence: inducer → pressure switch → igniter → gas valve → flame prove → blower.
  3. Measure, don’t guess. Replace only failed component.

Every branch should end with: probable cause, confirming test, and repair pricing path.

Superheat (Fixed Orifice)

  • Superheat = suction line temp - suction sat temp.
  • Compare measured to target chart (indoor WB + outdoor DB).

Subcooling (TXV)

  • Subcooling = liquid sat temp - liquid line temp.
  • Charge to manufacturer target where provided.

Static Pressure

  • TESP = return + supply static (magnitudes).
  • Compare to blower rated max external static.

Delta-T

  • Cooling split commonly ~16–22°F in many comfort systems.
  • Use as screening metric, not standalone diagnosis.
