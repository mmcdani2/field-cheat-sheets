Purpose: Diagnose electrical failures with meter data, not guesswork. This is the fastest path to fewer callbacks and cleaner closeout notes.

Safety First (Non-Negotiable)

  • Lockout/tagout where required.
  • Verify meter operation on known live source before and after testing.
  • Discharge capacitors safely before capacitance checks.
  • Use proper PPE and one-hand rule where applicable.

Core Electrical Workflow (Any No-Cool / No-Heat Call)

  1. Confirm call: Thermostat demand present (Y, G, W, O/B as applicable).
  2. Confirm high voltage: Line voltage at disconnect, contactor line side, and equipment lugs.
  3. Confirm control voltage: 24V path through safeties and board outputs.
  4. Test components: Capacitor, contactor coil/contacts, motors, board outputs, safeties.
  5. Validate result: Re-run system and capture amps/voltage/temps after repair.

Quick Test Targets

Component What to Check Field Pass/Fail Cue
Capacitor Measured µF vs label rating Replace if out of tolerance (commonly ±6% unless otherwise marked)
Contactor Coil 24V at coil on call 24V present but no pull-in = failed coil/mechanical bind
Contactor Contacts Voltage drop across closed contacts Excessive drop = pitted/high resistance contacts
Compressor/Fan Motor Voltage at motor + amp draw trend Correct voltage + no run = motor/start issue; high amps = load/restriction issue
Control Board Input present / output missing If correct input but no commanded output, board likely failed
Safeties (float, pressure, limits) Open/closed state under normal conditions Open safety breaks call path; verify root cause before bypass removal

Capacitor How-To

  1. Power off and verify zero volts.
  2. Discharge capacitor safely.
  3. Remove at least one lead to avoid backfeed reading.
  4. Measure µF (C-FAN and C-HERM on dual cap).
  5. Compare to label value and tolerance; replace if out.

If a capacitor failed, inspect motor amp draw and condenser cleanliness so the new one doesn’t fail again.

Contactor How-To

  1. On call, verify 24V across coil terminals.
  2. If coil energizes, check line/load voltage and contact condition.
  3. Measure voltage drop across closed contacts; high drop = resistance/heat risk.
  4. Inspect for ants/debris/pitting/welded contacts.

Motor Diagnostics (Condenser Fan / Blower)

  • Verify commanded voltage at motor terminals.
  • Check capacitor (PSC) or module/control signal (ECM).
  • Compare running amps to nameplate context (FLA/RLA where applicable).
  • Inspect for mechanical drag: blade rub, tight bearings, wheel debris, high static.

Control Voltage Fault Isolation (24V)

  1. Start at transformer secondary (~24V).
  2. Trace through thermostat call and safety chain.
  3. Find where voltage is lost (open switch, broken wire, failed board relay).
  4. Repair cause, then re-test full sequence.

Best practice: “Last good point / first bad point” method. It is fast and defensible.

Common Patterns and Likely Causes

  • Outdoor dead, indoor blower runs: open contactor coil circuit, float switch, pressure switch, board output, or disconnect issue.
  • Contactor chatters: weak low-voltage supply, loose connections, failing transformer, thermostat/control issue.
  • Breaker trips on startup: short to ground, locked rotor, failed compressor, pinched wiring, failed capacitor/start components.
  • Burned wire terminals: loose lugs/high resistance heating—repair terminal and root-cause current draw.

Quick Tips (New + Veteran Techs)

  • Take a photo of wiring before moving anything.
  • Tighten and re-check all replaced terminals—comeback prevention 101.
  • Don’t condemn a board until inputs, grounds, and safeties are verified.
  • Record before/after readings in notes: volts, amps, µF, and call path status.
  • If electrical symptom appears with airflow/refrigerant symptoms, solve systemically—not component by component.

Closeout Data to Log

  • Line voltage and control voltage readings
  • Capacitor measured µF (old/new)
  • Amp draw (compressor/fans/blower as applicable)
  • Failed component + confirming test result
  • Post-repair run verification (cool/heat confirmed)

Standard: No part replacement without a confirming measurement documented in the ticket.
