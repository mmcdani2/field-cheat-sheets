Purpose: Charge and validate TXV/EEV systems using measured subcooling versus manufacturer target.

When to Use

  • System controls evaporator superheat with TXV/EEV.
  • Nameplate/install docs provide target subcooling guidance.
  • System is running in stable cooling conditions with verified airflow.

How-To

  1. Confirm refrigerant type and OEM target subcooling.
  2. Measure liquid pressure at outdoor unit service port.
  3. Convert liquid pressure to saturation temperature (PT chart/app).
  4. Measure liquid line temperature near condenser outlet/service valve.
  5. Calculate: Subcooling = Liquid Saturation Temp - Liquid Line Temp.
  6. Compare measured SC to OEM target.
  7. Adjust in small increments, allow stabilization, then re-check SC and operating data.

Quick Tips

  • OEM target beats rule-of-thumb every time.
  • Dirty condenser coil or weak fan can fake an overcharge picture.
  • Log ambient, head pressure trend, and condenser airflow state with SC reading.
  • Verify line set length/diameter and install notes if readings seem inconsistent.

Common Failures

Symptom Likely Cause First Check
Subcooling low vs target Undercharge, flash gas, low condenser fill Leak indicators, charge history, line set issues, SH/SC pairing
Subcooling high vs target Overcharge, condenser-side backup, restricted heat rejection Condenser coil/fan condition, ambient/load context, non-condensable suspicion
SC swings rapidly Unstable operation, fan cycling anomalies, measurement error Probe contact/insulation, fan control behavior, trend over several minutes
Target SC achieved, comfort still poor Airflow/distribution/control issue TESP/static, Delta-T method consistency, zoning/damper behavior

Field rule: For TXV/EEV systems, charge to target subcooling only after condenser and airflow conditions are verified clean and stable.
