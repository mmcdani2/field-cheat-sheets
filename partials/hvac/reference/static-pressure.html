Purpose: Use TESP (Total External Static Pressure) to confirm airflow health before charge or component conclusions.

When to Use

  • No-cool/low-capacity/high-utility or comfort imbalance complaints.
  • Freeze-up callbacks or repeated part failures.
  • Before final charge adjustments or blower-speed decisions.

How-To

  1. Install proper test ports near equipment cabinet (return and supply sides, external to blower/coil as applicable).
  2. Measure return static (typically negative) and supply static (typically positive).
  3. Use magnitudes for total: TESP = |Return| + |Supply|.
  4. Compare TESP to equipment blower performance table / rated max external static.
  5. If TESP is high, locate dominant side (return vs supply) and isolate restrictions.

Quick Tips

  • Static is not optional—it's the gatekeeper for valid refrigerant diagnosis.
  • Same test locations every visit = comparable history.
  • Check filter, coil, dampers, and grilles before touching charge.
  • Document both total static and side split (return vs supply) for faster root cause isolation.

Common Failures

Symptom Likely Cause First Check
High return static Dirty filter, undersized return, blocked return path Filter status, return grille free area, closed doors/pressure imbalance
High supply static Dirty indoor coil, closed dampers/registers, undersized supply trunks Coil face condition, damper positions, branch restrictions
High total static (both sides elevated) System-wide airflow choke Blower setting/profile, duct design limits, accessory pressure drops
Low static but poor comfort Duct leakage/bypass, wrong fan profile, mixed air issues Duct integrity, blower configuration, return leakage pathways

Field rule: If static is out of range, fix airflow first—charge and Delta-T conclusions are low-confidence until then.
